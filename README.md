# CHZZK VOD CHAT

CHZZK VOD 채팅 수집 및 분석 프로그램입니다.

## 입력 규칙

- URL 최대 30개 혹은 파일명을 입력할 수 있습니다.
- 첫 번째 유효한 입력으로 실행 모드가 결정됩니다.
- URL이 먼저 입력되면 수집 모드
- 파일명이 먼저 입력되면 분석 모드
- 다른 형식의 입력은 자동으로 무시됩니다.

## 분석 기준

- 1분 단위 채팅 집계
- 키워드 포함 여부 기준 분석
- 이전 15분 이동평균
- 최소 채팅량 미만 구간은 증가율 및 키워드 랭킹에서 제외
- 키워드 개수/비율 TOP 및 상세 로그 출력

# 사용 방법

## 1. 채팅 수집

프로그램 실행 후 CHZZK VOD URL을 입력합니다. (최대 30개)

```
https://chzzk.naver.com/video/123456
https://chzzk.naver.com/video/654321

(Enter)
```

등록 내용을 확인한 뒤 채팅을 수집합니다.

프로그램 동일 경로에 폴더 및 파일이 생성됩니다.

```
chat/
└── 채널명/
    └── YYYY-MM-DD_영상번호.txt
```

---

## 2. 채팅 분석

프로그램 실행 후 분석할 단일 파일명을 입력합니다.

채팅 수집시 txt 파일명 양식과 동일해야합니다.

확장자는 입력하지 않아도 됩니다.

예시

```
2026-06-24_123456

(Enter)
```

분석할 시작 시간과 종료 시간을 입력합니다.

* 시간 형식: `HH:MM:SS`
* 시작 시간 미입력 시: `00:00:00`
* 종료 시간 미입력 시: 영상 종료 시간

분석할 키워드를 입력합니다. 기본 키워드는 `ㅋ` 입니다

또는 별개의 키워드를 입력해도 됩니다.

```
와
```

→ "와" 분석

분석 예정 정보를 확인한 뒤 계속할지 선택합니다.

생성 파일

```
chat/
└── 채널명/
    ├── YYYY-MM-DD_영상번호.txt
    └── YYYY-MM-DD_영상번호_analysis.txt
```
