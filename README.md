# ChzzkVODChat

치지직(CHZZK) VOD 채팅 로그 추출기

## 기능

- 치지직 VOD URL 입력
- 채팅 로그 자동 수집
- TXT 저장
- JSON 저장

## 사용법

1. Releases에서 ChzzkVODChat.zip 다운로드
2. 압축 해제 후 실행
3. VOD URL 입력 (스트리머의 동영상 항목의 영상 링크)

https://chzzk.naver.com/video/********

4. 완료 후 chat 폴더 확인

## 출력 형식

[00:00:00][닉네임] "채팅 내용"
