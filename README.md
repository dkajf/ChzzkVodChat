# CHZZK VOD CHAT

CHZZK VOD 채팅 수집 및 분석 프로그램입니다.

## 주요 기능

### VOD 채팅 수집 (v1.1)

- CHZZK VOD URL 입력
- 최대 10개 VOD 일괄 처리
- 채팅 JSON 저장
- 사람이 읽기 쉬운 TXT 저장
- 채널별 폴더 자동 정리

### 채팅 분석 (v1.2)

저장된 JSON 채팅 파일을 분석하여 방송 흐름을 빠르게 확인할 수 있습니다.

분석 항목

- 1분 단위 채팅량
- 키워드 포함 채팅 수
- 키워드 비율
- 이동평균(±10분) 기반 채팅 증가율
- 채팅량 TOP10
- 증가율 TOP10
- 키워드 TOP10
- 상세 로그 생성

분석 결과는 `_analysis.txt` 파일로 저장됩니다.

---

# 사용 방법

## 1. 채팅 수집

프로그램 실행 후 CHZZK VOD URL을 입력합니다.

```
https://chzzk.naver.com/video/123456
https://chzzk.naver.com/video/654321

(Enter)
```

등록 내용을 확인한 뒤 채팅을 수집합니다.

생성 파일

```
chat/
└── 채널명/
    ├── YYYY-MM-DD_영상번호.json
    └── YYYY-MM-DD_영상번호.txt
```

---

## 2. 채팅 분석

프로그램 실행 후 분석할 파일명을 입력합니다.

확장자는 입력하지 않아도 됩니다.

예시

```
2026-06-24_123456
2026-06-25_654321

(Enter)
```

등록된 파일을 확인한 뒤 키워드를 입력합니다.

```
Enter
```

→ 기본 키워드 `ㅋ`

또는

```
와
```

→ "와" 분석

생성 파일

```
chat/
└── 채널명/
    ├── YYYY-MM-DD_영상번호.json
    ├── YYYY-MM-DD_영상번호.txt
    └── YYYY-MM-DD_영상번호_analysis.txt
```

---

# 입력 규칙

- URL 또는 파일명을 최대 10개 입력할 수 있습니다.
- 첫 번째 유효한 입력으로 실행 모드가 결정됩니다.
- URL이 먼저 입력되면 수집 모드
- 파일명이 먼저 입력되면 분석 모드
- 다른 형식의 입력은 자동으로 무시됩니다.

---

# 분석 기준

- 1분 단위 채팅 집계
- 키워드 포함 여부 기준 분석
- 이동평균(±10분, 현재 분 제외)
- 방송 전체 평균 대비 비교
- TOP10 및 상세 로그 출력
