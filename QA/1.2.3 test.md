## 1. 테스트 목적

v1.2.3의 신규 기능과 개선 사항이 정상적으로 동작하는지 검증하고, 기존 분석 기능이 유지되는지 확인한다.

특히 TOP 랭킹 병합, 키워드 반복 제한, 평균 기반 TOP 후보 선정, ESC 종료 동작을 중점적으로 확인한다.

---

## 2. 테스트 범위

### Regression Test

* 기존 분석 기능 정상 유지 여부
* 채팅 분석 결과 저장
* 기존 출력 포맷 유지 여부

### Functional Test

* TOP 랭킹 병합 동작
* 병합 후 인접 구간 재검사
* 최대 3개 시간대 병합 제한
* 병합 구간 시간순 출력
* 키워드 반복 제한
* 전체 평균 채팅 수 기준 TOP 후보 선정
* 메인 메뉴 ESC 종료

### Validation Test

* 병합이 필요 없는 경우 유지 여부
* 병합 최대 개수 초과 시 처리 방식
* 키워드 반복이 5회 초과된 경우 카운트 방식
* 평균 이하 채팅량 구간이 TOP에서 제외되는지 여부

## 4. 테스트 케이스

### TC-601: 프로그램 실행

* 예상 결과
  * 프로그램이 정상 실행된다.

### TC-602: 기존 분석 기능 유지 확인

* 예상 결과
  * 기존 채팅 분석 흐름이 정상적으로 동작한다.
  * 분석 결과 파일이 정상 생성된다.

### TC-603: TOP 랭킹 병합 확인

* 테스트 항목
  * 연속된 시간대의 TOP 항목이 동일 장면으로 판단되는 경우
* 예상 결과
  * 해당 시간대가 하나의 병합 구간으로 표시된다.

### TC-604: 병합 재검사 확인

* 테스트 항목
  * 병합된 구간과 인접한 다른 시간대가 다시 병합 대상이 되는 경우
* 예상 결과
  * 인접 구간이 재검사되어 최종 병합 결과로 반영된다.

### TC-605: 병합 최대 개수 제한 확인

* 테스트 항목
  * 4개 이상의 연속 시간대가 후보가 되는 경우
* 예상 결과
  * 최대 3개 시간대까지만 병합되며 나머지는 유지된다.

### TC-606: 병합 구간 시간순 출력 확인

* 예상 결과
  * 병합된 구간이 시간순(오름차순)으로 출력된다.

### TC-607: 키워드 반복 제한 확인

* 테스트 항목
  * 한 채팅에 동일 키워드가 6회 이상 포함된 경우
* 예상 결과
  * 해당 키워드는 최대 5회만 카운트된다.

### TC-608: 다른 채팅에 대한 영향 없음 확인

* 예상 결과
  * 다른 채팅의 키워드 카운트에는 영향을 주지 않는다.

### TC-609: 평균 기반 TOP 후보 선정 확인

* 예상 결과
  * 전체 평균 채팅 수 미만인 시간대는 TOP 후보에서 제외된다.

### TC-610: 메인 메뉴 ESC 종료 확인

* 예상 결과
  * 메인 메뉴에서 ESC를 누르면 프로그램이 즉시 종료된다.

### TC-611: 분석 결과 파일 생성 확인

* 예상 결과
  * `_analysis.txt` 파일이 정상 생성된다.

---

## 5. 예상 결과

* TOP 랭킹이 더 직관적이고 실제 하이라이트 흐름에 가깝게 표시된다.
* 키워드 분석이 과도한 반복으로 왜곡되지 않는다.
* 평균 이하의 불필요한 구간이 TOP에서 제외되어 결과가 더 깔끔해진다.
* 메인 메뉴에서 ESC로 빠르게 종료할 수 있다.

# 7. 향후 개발 계획

v1.2.3 이후에는 분석 결과의 시각화와 사용자 경험을 더 향상시키기 위한 기능 검토를 진행한다.

예정 기능

* 병합 기준을 사용자 설정 가능하도록 개선
* 다중 키워드 분석 지원
* 분석 결과 요약 리포트 자동 생성
* TOP 랭킹 시각화 기능 추가
