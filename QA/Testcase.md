│
├─ 설정 파일 확인
│   ├─ 설정 존재
│   │   └─ 설정 불러오기
│   └─ 설정 없음
│       └─ 기본 설정 생성
│
├─ 메인 메뉴
│   │
│   ├─ 사용자 입력 확인
│   │   │
│   │   ├─ URL 입력 → 채팅 수집 모드
│   │   │   │
│   │   │   ├─ 수집 정보 확인
│   │   │   ├─ 채팅 수집 시작
│   │   │   │   └─ 진행률 표시
│   │   │   ├─ TXT 파일 생성
│   │   │   └─ 메인 메뉴 복귀
│   │   │
│   │   ├─ TXT 파일명 입력 → 채팅 분석 모드
│   │   │   │
│   │   │   ├─ 분석 구간 입력
│   │   │   │   ├─ 시작 시간
│   │   │   │   └─ 종료 시간
│   │   │   ├─ 키워드 입력
│   │   │   ├─ 분석 정보 확인

│   │   │   │   ├─ 채팅량 TOP
│   │   │   │   ├─ 이동평균 TOP

│   │   │   │   ├─ 키워드 비율 TOP
│   │   │   │   └─ 상세 로그
│   │   │   └─ 메인 메뉴 복귀
│   │   │
│   │   └─ ESC 입력 → 프로그램 종료
│
└─ 프로그램 종료

## 1. 프로그램 실행

```
- 프로그램
프로그램이 정상적으로 실행되는가
중복 실행 시 정상적으로 동작하는가
프로그램 작업 폴더를 정상적으로 생성하는가
작업 폴더 생성에 실패할 경우 오류를 안내하는가
필요한 하위 폴더(chat 등)가 존재하지 않을 경우 자동 생성하는가
실행 중 예외가 발생하여도 프로그램이 비정상 종료되지 않는가
콘솔 창의 레이아웃이 정상적으로 출력되는가
프로그램 제목 및 버전 정보가 정상적으로 표시되는가
메인 메뉴가 정상적으로 출력되는가
입력 대기 상태가 정상적으로 활성화되는가
아무 입력 없이 Enter를 누를 경우 적절히 처리하는가
ESC 입력 시 프로그램이 정상 종료되는가
종료 시 리소스를 정상적으로 정리하는가
재실행 시 이전 실행의 영향 없이 정상적으로 시작되는가
실행 위치(작업 디렉터리)가 달라도 정상적으로 동작하는가

- 설정 파일
설정 파일이 존재하지 않을 경우 기본 설정을 생성하는가
설정 파일이 손상되어 있을 경우 예외 처리하는가
설정 파일의 필수 항목이 누락되어 있을 경우 복구 또는 기본값을 적용하는가
설정 파일을 정상적으로 불러오는가
```

## 2. 채팅 수집 기능

### 2.1 입력

```
- 입력
URL을 정상적으로 입력받는가
치지직 URL 형식이 아닐 경우 안내하는가
VOD URL이 아닌 경우 안내하는가
존재하지 않거나 삭제, 비공개된 VOD 번호, 혹은 라이브 URL을 입력할 경우 안내하는가
URL 앞뒤 공백을 제거하여 처리하는가
URL 뒤에 불필요한 파라미터가 포함되어 있어도 정상 처리하는가
여러 개의 URL을 줄바꿈으로 입력할 경우 모두 인식하는가
중복된 URL을 입력할 경우 적절히 처리하는가
빈 입력 시 메인 메뉴로 복귀하거나 안내하는가
공백만 입력한 경우 적절히 처리하는가
허용되지 않은 문자열을 입력할 경우 오류를 안내하는가
매우 긴 문자열을 입력하여도 비정상 종료되지 않는가
(y/n)
수집 시작 전 VOD 정보를 정상적으로 조회하는가

- 확인
수집 정보 확인 화면이 정상적으로 표시되는가
수집 정보 확인 후 취소가 가능한가
ESC 입력 시 수집을 취소할 수 있는가
```

### 2.2 출력

```
- 진행
채팅 수집이 정상적으로 시작되는가
진행률이 정상적으로 갱신되는가
현재 수집 개수가 정상적으로 표시되는가
총 수집 개수가 실제 채팅 수와 일치하는가
채팅이 없는 구간도 정상적으로 처리하는가
17시간 길이의 VOD도 정상적으로 수집되는가
짧은 VOD도 정상적으로 수집되는가
수집 중 네트워크 오류 발생 시 적절히 처리하는가
수집 중 일시적인 API 오류 발생 시 복구 가능한가
수집 중 ESC 입력 시 안전하게 종료되는가

- 출력
수집 완료 후 TXT 파일이 생성되는가
TXT 파일명이 올바르게 생성되는가
동일한 파일명이 존재할 경우 적절히 처리하는가
생성된 TXT 파일의 인코딩이 UTF-8로 저장되는가
채팅 시간이 정상적으로 기록되는가
채팅 내용이 누락 없이 저장되는가
이모지 및 한글이 정상적으로 저장되는가
특수문자가 포함된 채팅도 정상적으로 저장되는가
빈 채팅이 저장되지 않는가
수집 완료 메시지가 정상적으로 출력되는가
수집 완료 후 메인 메뉴로 정상 복귀하는가
```

## 3. 채팅 분석 기능

### 3.1 입력

```
TXT 파일명을 정상적으로 입력받는가
존재하지 않는 TXT 파일명을 입력할 경우 안내하는가
TXT가 아닌 파일명을 입력할 경우 안내하는가
빈 입력 시 적절히 처리하는가
공백만 입력한 경우 적절히 처리하는가
파일명이 대소문자와 관계없이 정상적으로 처리되는가
손상된 TXT 파일을 입력할 경우 오류를 안내하는가
분석 가능한 채팅 데이터인지 확인하는가
시작 시간을 정상적으로 입력받는가
종료 시간을 정상적으로 입력받는가
시간 형식(HH:MM:SS)이 올바르지 않을 경우 안내하는가
시작 시간이 종료 시간보다 늦을 경우 안내하는가
방송 길이를 초과하는 시간을 입력할 경우 안내하는가
음수 또는 허용되지 않은 시간을 입력할 경우 안내하는가
키워드를 정상적으로 입력받는가
빈 키워드를 입력할 경우 안내하는가
공백만 입력한 키워드를 적절히 처리하는가
특수문자 키워드를 정상적으로 처리하는가
한글 키워드를 정상적으로 처리하는가
영문 및 숫자 키워드를 정상적으로 처리하는가
분석 정보 확인 화면이 정상적으로 표시되는가
분석 시작 전 취소가 가능한가
ESC 입력 시 메인 메뉴로 복귀하는가
```

채팅 분석이 정상적으로 시작되는가

분석 완료 후 메인 메뉴로 정상 복귀하는가
