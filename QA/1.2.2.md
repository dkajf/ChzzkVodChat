# CHZZK VOD Chat Analyzer v1.2.2 개발 이력

## 1. 버전 개요

### 버전

v1.2.2

### 개발 목적

v1.2.1에서 채팅 분석 기능을 안정화한 이후, 사용자가 프로그램을 연속으로 편리하게 사용할 수 있도록 UI와 분석 흐름을 개선하는 데 중점을 두었다.

이번 버전에서는 채팅 분석 모드를 단일 파일 단위로 단순화하고, 원하는 시간대만 분석할 수 있도록 하여 사용자 편의성을 높이고 불필요한 반복 실행을 줄이는 것이 목적이다.

---

# 2. 개발 배경

v1.2.1까지는 채팅 분석이 완료되면 프로그램이 종료되었고, 분석 시 다중 파일 입력 방식으로 사용자가 불편을 겪을 수 있었다.

또한 긴 방송에서 전체 채팅을 분석하는 방식은 필요한 구간만 빠르게 확인하려는 사용자 요구를 충분히 반영하지 못했다.

이에 따라 연속 작업성을 확보하고, 시간대 선택 기반 분석 기능을 도입하여 사용 시간을 단축시키는 개선을 진행하였다.

---

# 3. 개선 사항

## 3.1 메인 메뉴 복귀 기능 개선

### 개선 전

* 채팅 수집 또는 분석 완료 후 프로그램이 즉시 종료되었다.
* 다른 작업을 위해 프로그램을 다시 실행해야 했다.

### 개선 후

* 모든 작업 완료 후 메인 메뉴로 자동 복귀하도록 변경하였다.
* 사용자는 채팅 수집 및 분석을 연속으로 수행할 수 있다.

---

## 3.2 채팅 분석 모드 UI 및 입력 흐름 개선

### 개선 전

* 파일명 입력 → 키워드 입력 → 분석
* 다중 파일 입력 모드가 제공되었다.

### 개선 후

* 파일명 입력 → 분석 시작 시간 입력 → 분석 종료 시간 입력 → 키워드 입력 → 분석 정보 확인 → 분석 시작
* 분석 시작/종료 시간 미입력 시 기본값을 적용하도록 변경하였다.
* 분석 대상은 항상 단일 파일로 제한하였다.

---

## 3.3 시간 입력 검증 강화

### 개선 내용

* `HH:MM:SS` 형식이 아닌 입력은 오류 처리한다.
* 시작 시간이 종료 시간보다 늦으면 다시 입력 받는다.
* 영상 길이를 초과하는 시간 입력은 오류 처리한다.
* 음수 시간 입력은 오류 처리한다.
* 오류 발생 시 안내 메시지를 출력하고 해당 입력을 재요청한다.

---

## 3.4 다중 파일 분석 모드 제거

### 개선 내용

* 기존 다중 파일 분석 모드를 제거하여 분석 과정이 단순해졌다.
* 분석은 항상 하나의 TXT/JSON 파일만 대상으로 수행한다.

# 5. 기대 효과

* 프로그램을 재실행하지 않고 연속 작업이 가능해진다.
* 원하는 시간대만 분석하여 분석 시간을 단축할 수 있다.
* 긴 방송에서도 필요한 구간만 빠르게 확인할 수 있다.
* 입력 과정이 단순해져 사용자 편의성이 향상된다.
