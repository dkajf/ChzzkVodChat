# 1. 테스트 목적

v1.0에서 구현된 핵심 기능을 유지하면서 실제 사용 과정에서 발견된 문제를 개선하였는지 검증한다.

또한 장시간 VOD 수집 환경에서도 데이터의 중복, 누락 없이 안정적으로 저장되는지 확인한다.

# 2. 테스트 범위

### Regression Test

* 프로그램 실행
* 방송 조회

* TXT 저장

### Functional Test

* 방송 정보 출력
* 진행 로그
* 저장 완료 메시지

### Data Integrity Test

* 마지막 페이지 처리

* JSON ↔ TXT 일치

### Validation Test

* URL 검증
* 존재하지 않는 방송
* 예외 처리

### Usability Test

* 진행률 확인
* 로그 가독성
* 오류 메시지

# 9. 실제 사용 중 확인한 개선 사항

이번 테스트 과정에서 다음 사항을 차기 버전 개선 대상으로 기록하였다.

* 장시간 방송에서 동일 이벤트가 여러 분에 걸쳐 반복 추천되는 문제
* 채팅량이 적은 구간이 증가율만으로 상위에 노출되는 현상
* 채팅량, 증가율, 키워드를 각각 따로 분석하여 실제 하이라이트 선정 효율이 떨어지는 문제
* 분석 결과에서 연속된 이벤트를 하나의 구간으로 병합할 필요성
* 이동 평균을 활용한 증가율 계산 방식 검토 필요

# 11. 테스트 결론

v1.1은 기존 핵심 기능을 유지하면서 데이터 수집 안정성과 사용성을 향상시킨 버전이다.

Regression Test를 통해 기존 기능이 정상적으로 유지됨을 확인하였으며, Data Integrity Test를 통해 치지직 API의 페이지 기반 채팅 수집 과정에서 데이터의 중복 및 누락이 발생하지 않음을 검증하였다.

또한 실제 장시간 VOD를 이용한 사용성 테스트를 수행한 결과, 하이라이트 분석 기능의 정확도를 더욱 높이기 위해 이벤트 점수 기반 분석과 연속 이벤트 병합 기능을 차기 버전(v1.2)의 개선 과제로 선정하였다.
