# 6. 테스트 계획

## 문서 정보

| 항목     | 내용                                                                 |
| ------ | ------------------------------------------------------------------ |
| 프로젝트   | CHZZK VOD Chat Analyzer                                              |
| 버전     | v1.2.2                                                              |
| 문서번호   | TP-005                                                              |
| 작성자    | 오세현                                                                 |
| 테스트 유형 | Regression Test / Functional Test / Validation Test / Usability Test |

---

## 1. 테스트 목적

v1.2.2의 개선 사항이 정상적으로 동작하는지 검증하고, 기존 채팅 분석 기능이 유지되는지 확인한다.

특히 분석 시간대 선택, 입력 검증, 메인 메뉴 복귀 동작이 요구사항에 맞게 작동하는지 중점적으로 확인한다.

---

## 2. 테스트 범위

### Regression Test

* 기존 분석 기능 정상 유지 여부
* 파일명 입력 및 대상 파일 조회
* 분석 결과 저장

### Functional Test

* 시작 / 종료 시간 입력 흐름
* 시간 입력 검증
* 키워드 입력 및 분석 진행
* 분석 정보 확인 후 진행/취소
* 메인 메뉴 복귀 동작

### Validation Test

* 형식이 잘못된 시간 입력
* 시작 시간이 종료 시간보다 늦은 경우
* 영상 길이 초과 시간 입력
* 음수 시간 입력

## 4. 테스트 케이스

### TC-501: 프로그램 실행

* 예상 결과
  * 프로그램이 정상 실행된다.

### TC-502: 단일 파일 분석 모드 진입

* 예상 결과
  * 파일명 입력 후 분석 모드로 진입한다.

### TC-503: 시작 시간 미입력 처리

* 예상 결과
  * 시작 시간이 자동으로 `00:00:00`으로 설정된다.

### TC-504: 종료 시간 미입력 처리

* 예상 결과
  * 종료 시간이 영상 종료 시간으로 설정된다.

### TC-505: 시간 형식 검증

* 테스트 항목
  * `0:00:00`, `00:0:00`, `abcd`, `00:60:00` 등
* 예상 결과
  * 모두 오류 메시지가 출력되며 재입력을 요청한다.

### TC-506: 시작 시간이 종료 시간보다 늦은 경우

* 예상 결과
  * 오류 메시지가 출력되고 재입력을 요청한다.

### TC-507: 영상 길이 초과 시간 입력

* 예상 결과
  * 오류 메시지가 출력되고 재입력을 요청한다.

### TC-508: 음수 시간 입력

* 예상 결과
  * 오류 메시지가 출력하고 재입력을 요청한다.

### TC-509: 키워드 입력 및 분석 정보 확인

* 예상 결과
  * 분석 예정 정보가 화면에 출력된다.
  * 사용자가 `Y` 선택 시 분석을 시작한다.
  * `N` 선택 시 메인 메뉴로 이동한다.

### TC-510: 메인 메뉴 복귀 확인

* 예상 결과
  * 분석 완료 후 메인 메뉴로 복귀한다.

### TC-511: 분석 결과 저장 확인

* 예상 결과
  * `_analysis.txt` 결과 파일이 정상 생성된다.

---

## 5. 예상 결과

* 사용자는 분석 구간을 직관적으로 입력할 수 있다.
* 시간 검증 로직으로 잘못된 입력을 방지할 수 있다.
* 분석 완료 후 메인 메뉴로 자연스럽게 돌아온다.
* 분석 대상 파일이 하나로 제한되어 UI가 간결해진다.

---

# 7. 향후 개발 계획

v1.2.2 이후에는 분석 정확도와 활용성을 더욱 높이기 위한 추가 기능 검토를 진행한다.

예정 기능

* 분석 구간을 파일별로 저장하는 기능
* 다중 키워드 입력 및 복합 분석
* 분석 요약 보고서 자동 생성
* 분석 대상 구간 시각화
