# CHZZK VOD Chat v1.2 개발 이력

## 1. 버전 개요

### 버전

v1.2

### 개발 목적

v1.1에서 기능 안정성을 확보한 이후, 프로그램의 유지보수성과 배포 편의성을 향상시키기 위한 개선 작업을 진행하였다.

이번 버전에서는 신규 기능 추가보다는 프로젝트 구조 개선, 실행 파일 배포, 문서화 및 사용자 경험 향상에 중점을 두었다.

---

# 2. 개발 배경

v1.1까지의 테스트를 통해 핵심 기능은 안정적으로 동작하는 것을 확인하였다.

그러나 프로그램을 다른 환경에서도 쉽게 사용할 수 있도록 하기 위해 다음과 같은 개선 필요 사항이 확인되었다.

* Python 설치 없이 실행 가능한 배포 파일 필요
* 프로젝트 구조 정리 필요
* 사용 방법 문서 부족
* 출력 메시지 가독성 향상 필요
* 유지보수를 위한 코드 정리 필요

이에 따라 배포 환경 구축 및 유지보수성 향상을 목표로 개선 작업을 진행하였다.

---

# 3. 개선 사항

## 3.1 프로젝트 구조 개선

### 개선 전

소스 코드와 배포 파일이 혼재되어 프로젝트 관리가 어려웠다.

### 개선 후

프로젝트 디렉터리를 기능별로 정리하여 유지보수성과 가독성을 향상시켰다.

---

## 3.2 코드 리팩터링

### 개선 전

일부 코드의 중복과 가독성이 떨어지는 부분이 존재하였다.

### 개선 후

중복 코드를 정리하고 함수 구조를 개선하여 유지보수가 용이하도록 수정하였다.

---

## 3.3 실행 파일(EXE) 배포 지원

### 개선 전

Python 환경에서만 실행 가능하였다.

### 개선 후

PyInstaller를 이용하여 실행 파일을 생성하고, Python이 설치되지 않은 환경에서도 사용할 수 있도록 배포 환경을 구성하였다.

---

## 3.4 프로젝트 문서화

### 개선 전

프로그램 사용 방법이 별도로 정리되어 있지 않았다.

### 개선 후

README를 작성하여 설치 방법, 실행 방법 및 주요 기능을 문서화하였다.

---

## 3.5 GitHub 저장소 정리

### 개선 전

빌드 과정에서 생성되는 불필요한 파일이 저장소에 포함될 가능성이 있었다.

### 개선 후

저장소 구조를 정리하고 배포에 필요한 파일만 관리하도록 개선하였다.

---

## 3.6 사용자 안내 메시지 개선

### 개선 전

일부 콘솔 출력이 일관되지 않아 현재 진행 상태를 파악하기 어려웠다.

### 개선 후

출력 형식을 통일하여 시작, 진행, 완료 및 오류 상태를 쉽게 확인할 수 있도록 개선하였다.

# 9. 실제 사용 중 발견한 개선 사항

10개의 실제 방송을 분석한 결과 다음과 같은 개선 필요 사항을 확인하였다.

### AI-001

증가율만 높은 구간이 실제 하이라이트보다 우선 추천되는 현상

예)

채팅 12개 → 증가율 500%

---

### AI-002

동일 이벤트가 여러 분에 걸쳐 TOP10에 중복 노출되는 현상

예)

05:27

05:28

05:29

05:30

---

### AI-003

채팅량·증가율·키워드가 각각 독립적으로 계산되어 실제 중요 이벤트를 종합적으로 판단하기 어려움

---

### AI-004

증가율 계산이 직전 1분 기준이라 작은 변동에도 민감하게 반응하는 현상

---

### AI-005

# 12. 테스트 결론

v1.2에서는 채팅 분석 기능이 정상적으로 동작하며, 다양한 실제 VOD 데이터를 대상으로 안정적으로 분석 결과를 생성함을 확인하였다.

또한 실제 방송 10건 이상을 대상으로 검증을 수행한 결과, 기능적인 오류는 발견되지 않았으나 분석 알고리즘의 정확도를 더욱 향상시킬 수 있는 개선 사항을 도출하였다.

해당 개선 사항은 차기 버전(v1.3)의 알고리즘 개선 요구사항으로 관리한다.
