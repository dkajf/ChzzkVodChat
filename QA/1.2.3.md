# CHZZK VOD Chat Analyzer v1.2.3 개발 이력

## 1. 버전 개요

### 버전

v1.2.3

### 개발 목적

v1.2.2까지의 분석 기능을 기반으로, TOP 랭킹의 직관성과 분석 품질을 향상시키는 데 중점을 두었다.

이번 버전에서는 동일 장면이 여러 순위에 중복 표시되는 문제와, 동일 채팅에서 키워드가 과도하게 반복되어 분석 결과가 왜곡되는 문제를 해소한다. 또한 메인 메뉴에서 빠르게 종료할 수 있는 단축키와 TOP 후보 선정 기준 개선을 적용한다.

---

# 2. 개발 배경

기존 분석 결과에서는 동일한 장면이 연속된 시간대에 여러 번 등장해 TOP 랭킹이 분산되는 문제가 있었다.

또한 하나의 채팅에서 동일 키워드를 과도하게 반복하는 경우, 키워드 통계가 실제 흐름을 반영하지 못하고 왜곡될 수 있었다.

이와 함께 TOP 랭킹의 최소 채팅 기준이 사용자 임의 설정값에 의존하던 구조로, 분석 결과의 기준이 불명확하게 느껴질 수 있었다.

이에 따라 TOP 랭킹의 병합 기준, 키워드 반복 제한, 평균 기반 후보 선정, 메뉴 종료 단축키를 추가하여 분석의 신뢰성과 사용성을 높이는 개선을 진행하였다.

---

# 3. 개선 사항

## 3.1 TOP 랭킹 클립 병합

### 개선 전

* 동일한 장면이 여러 순위에 중복 표시될 수 있었다.
* TOP 랭킹이 실제 하이라이트 기준이 아니라 단순 시간대 분산으로 보일 수 있었다.

### 개선 후

* 연속된 시간대의 TOP 항목을 병합하여 보다 자연스러운 하이라이트 기준의 순위를 제공한다.
* 병합 기준은 ±1분 차이이며, 최대 3개 시간대까지 허용한다.
* 병합된 구간은 시간순으로 출력된다.

---

## 3.2 키워드 반복 카운트 제한

### 개선 전

* 한 채팅에서 동일 키워드가 반복될 경우 그 횟수만큼 과도하게 반영될 수 있었다.

### 개선 후

* 한 채팅 내 동일 키워드는 최대 5회까지만 카운트한다.
* 초과되는 반복은 무시된다.
* 다른 채팅에는 영향을 주지 않는다.

---

## 3.3 TOP 최소 채팅 기준 변경

### 개선 전

* TOP 후보 선정 기준이 사용자 지정 최소 채팅 수에 의존하였다.

### 개선 후

* 전체 평균 채팅 수 이상인 시간대만 TOP 후보로 사용한다.
* 평균보다 적은 채팅량의 구간은 TOP 랭킹에서 제외된다.
* 이동평균 증가율 및 기타 분석에는 영향을 주지 않는다.

---

## 3.4 메인 메뉴 ESC 종료 지원

### 개선 전

* 메인 메뉴에서 종료하려면 프로그램을 직접 종료해야 했다.

### 개선 후

* 메인 메뉴 입력 대기 상태에서 ESC를 누르면 프로그램이 즉시 종료된다.

# 4. 변경 사항 요약

| 구분 | 내용 |
| ---- | ---- |
| 기능 개선 | TOP 랭킹 연속 시간대 병합 |
| 기능 개선 | 병합 후 인접 시간대 재검사 |
| 기능 개선 | 최대 3개 시간대 병합 제한 |
| 기능 개선 | 병합 구간 시간순 출력 |

| 기능 개선 | TOP 최소 채팅 컷을 전체 평균 채팅 수 기준으로 변경 |
| 기능 개선 | 메인 메뉴 ESC 종료 지원 |

---

# 5. 기대 효과

* TOP 랭킹이 실제 하이라이트 흐름에 더 가깝게 표시된다.
* 키워드 분석이 과도한 반복으로 왜곡되지 않는다.
* 평균 채팅량 이하의 불필요한 구간이 TOP에서 제외되어 결과가 더 깔끔해진다.
* 메인 메뉴에서 빠르게 종료할 수 있어 사용성이 향상된다.
