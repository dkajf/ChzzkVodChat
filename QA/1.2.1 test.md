# 8. 실제 검증 결과

실제 VOD 10개를 대상으로 분석한 결과 다음 사항을 확인하였다.

* 동일 이벤트가 하나의 추천으로 병합됨
* 추천 결과의 다양성이 향상됨
* 증가율만 높은 작은 이벤트의 비중 감소
* 채팅량 기반 추천 정확도 향상
* 실제 하이라이트와의 일치율 향상

---

# 9. 차기 개선 사항

이번 테스트 과정에서 다음 요구사항을 도출하였다.

### AR-001

이벤트 점수(Event Score) 도입

채팅량, 증가율 및 키워드를 하나의 점수로 계산하여 추천 정확도를 향상한다.

---

### AR-002

이동 평균 기반 증가율 계산

직전 1분이 아닌 이동 평균을 이용하여 증가율을 계산한다.

---

### AR-003

최소 채팅량 조건

채팅량이 적은 구간은 증가율이 높더라도 추천 대상에서 제외한다.

---

### AR-004

복수 키워드 지원

'ㅋ' 외에도 다양한 반응 키워드를 분석 대상으로 추가한다.

# 11. 테스트 결론

v1.2.1에서는 분석 알고리즘 개선을 통해 실제 하이라이트 추천 품질이 향상되었음을 확인하였다.

Regression Test 결과 기존 기능은 모두 정상 동작하였으며, Recommendation Quality Test를 통해 동일 이벤트 중복 추천이 감소하고 추천 결과의 활용성이 개선되었다.

향후에는 이벤트 점수 기반 추천 알고리즘을 도입하여 분석 정확도를 더욱 향상시킬 예정이다.
