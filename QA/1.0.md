# CHZZK VOD Chat 개발 배경 및 버전 이력 (v1.0 ~ v1.0.2)

## 1. 프로젝트 개요

### 프로젝트명

CHZZK VOD Chat

### 개발 목적

치지직(CHZZK) VOD에서 채팅 데이터를 수집하여 JSON 및 TXT 파일 형태로 저장하는 프로그램이다.

기존에는 VOD의 특정 장면을 찾기 위해 영상을 직접 탐색해야 했으며, 방송 시간이 긴 경우 원하는 구간을 찾는 데 많은 시간이 소요되었다.

또한 채팅 반응을 분석하기 위해서는 수작업으로 채팅을 확인해야 했기 때문에 반복 작업이 발생하였다.

이를 해결하기 위해 다음과 같은 목표를 가지고 프로그램을 개발하였다.

* 치지직 VOD 채팅 자동 수집
* JSON 및 TXT 형태로 저장
* 이후 채팅 분석 프로그램에서 사용할 수 있는 데이터 생성
* 반복 작업 최소화

---

# 2. 개발 목표

## v1.0 목표

* VOD URL 입력
* 방송 정보 조회
* 채팅 데이터 수집
* JSON 저장
* TXT 저장

"사용자가 VOD 하나의 채팅을 문제없이 추출할 수 있는 최소 기능 구현"을 목표로 한다.

---

# 3. v1.0 구현 기능

### 3.1 URL 입력

사용자가 치지직 VOD URL을 입력할 수 있다.

---

### 3.2 방송 정보 조회

입력된 URL에서

* 방송 제목
* 방송 ID
* 방송 길이

등의 정보를 조회한다.

---

### 3.3 채팅 수집

치지직 API를 이용하여

전체 VOD의 채팅 데이터를 순차적으로 수집한다.

---

### 3.4 JSON 저장

원본 데이터를 JSON 형태로 저장한다.

---

### 3.5 TXT 저장

JSON 데이터를 사람이 읽기 쉬운 TXT 형태로 변환하여 저장한다.

---

### 3.6 저장 폴더 생성

저장 폴더가 존재하지 않을 경우 자동 생성한다.

---

### 3.7 진행 상황 출력

콘솔에서

* 현재 수집 개수
* 진행 상황

을 출력하여 사용자가 작업 진행 여부를 확인할 수 있도록 한다.
