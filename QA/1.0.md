# CVC_v1.0 개발 이력

## 1. 개요

### 프로젝트명

CHZZK VOD Chat

### 개발 목적

치지직(CHZZK) 다시보기 VOD에서 채팅 데이터를 수집하여 JSON 및 TXT 파일 형태로 저장하는 프로그램이다.

채팅 전체를 수집하여 반응을 분석, 수월한 클립 생성이라는 목표를 가지고 프로그램을 개발하였다.

* 치지직 VOD 채팅 자동 수집
* JSON 및 TXT 형태로 저장
* 이후 채팅 분석 프로그램에서 사용할 수 있는 데이터 생성

## 2. v1.0 목표

* VOD URL 입력
* 방송 정보 조회
* 채팅 데이터 수집
* JSON, TXT 저장

"VOD 하나의 채팅을 추출할 수 있는 최소 기능 구현"을 목표

## 3. v1.0 구현 기능

- URL 입력

    사용자가 치지직 다시보기 URL을 입력할 수 있다.
- 방송 정보 조회

    입력된 URL에서 채팅 수집 및 가공에 필요한 방송 제목, ID, 시간을 조회한다.
- 3.3 채팅 수집

    치지직 API를 이용하여 전체 VOD의 채팅 데이터를 순차적으로 수집한다.

- 3.4 저장

    원본 데이터를 JSON 형태로 저장 후 읽기편한 TXT로 변환 가공하여 저장한다.
    이때 저장 폴더가 없을 시 자동 생성한다.

- 3.5 진행 상황 출력

    콘솔에서 현재 채팅 수집 개수, 진행 상황을 을 출력하여 사용자가 작업 진행 여부를 확인할 수 있도록 한다.

## 4. 확인된 불편 사항

실제 프로그램을 사용하며 다음과 같은 개선점이 확인되었다.

- 예외 처리 부족

    잘못된 URL 입력 시 사용자에게 원인을 충분히 안내하지 못하였다.

-  파일 저장 안정성

    방송 제목에 사용할 수 없는 특수문자가 포함될 경우 파일 저장 오류가 발생할 가능성이 있었다.

- 4.3 진행 상태 확인의 불편함

    장시간 방송에서는 현재 어느 정도 수집되었는지 직관적으로 확인하기 어려웠다.

- 4.4 결과 확인 불편

    수집 완료 후 생성된 파일 위치를 사용자가 직접 찾아야 했다.

- 4.5 오류 원인 확인 어려움

    예외 발생 시 어떤 단계에서 실패했는지 확인하기 어려웠다.
