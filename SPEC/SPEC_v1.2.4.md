# CHZZK VOD CHAT v1.2.4 Specification

## 설정 파일 시스템 추가

### 목적

프로그램 실행 시 설정 파일을 자동으로 관리하여 사용자가 기본 옵션을 직접 수정할 수 있도록 한다.

---

## 설정 파일

### 파일명

```
CHZZK_VOD_CHAT_Settings.txt
```

프로그램 실행 파일과 동일한 폴더에 생성된다.

---

## 최초 실행

프로그램 실행 시 설정 파일 존재 여부를 확인한다.

* 설정 파일이 존재하지 않을 경우 기본 설정 파일을 생성한다.
* 생성 완료 후 메인 메뉴 진입 시 다음 안내를 출력한다.

```
설정 파일(CHZZK_VOD_CHAT_Settings.txt)이 생성되었습니다.
```

---

## 프로그램 실행 순서

```
프로그램 실행

↓

설정 파일 존재 확인

↓

(없으면 생성)

↓

설정 파일 읽기

↓

설정값 검증

↓

메인 메뉴
```

## 설정 파일 구조

```
CHZZK_VOD_CHAT_Settings.txt
```

```ini
# ==========================================
# CHZZK VOD CHAT Settings
# ==========================================

ConfigFormat=1

[Collect]

; 한 번에 등록할 수 있는 최대 VOD 개수입니다.
; 권장 범위 : 1 ~ 30
MaxVods=30

; 채팅 파일(.json/.txt)이 저장될 폴더입니다.
; 프로그램 실행 폴더를 기준으로 생성됩니다.
OutputFolder=chat


[Analyze]

; 분석 실행 시 기본으로 사용할 키워드입니다.
; 입력 없이 Enter를 누르면 이 키워드가 사용됩니다.
DefaultKeyword=ㅋ

; 하나의 채팅에서 동일 키워드를 최대 몇 번까지 카운트할지 설정합니다.
; 0으로 설정하면 제한 없이 모두 카운트합니다.
MaxKeywordRepeat=5

; TOP 구간에 포함되기 위한 최소 채팅 수를 설정합니다.
; 0으로 설정하면 방송 전체 평균 채팅 수를 자동으로 사용합니다.
TopMinimumChatCount=0

; 이동 평균 계산에 사용할 이전 구간(분)입니다.
; 값이 클수록 평균이 부드럽게 계산됩니다.
MovingAverageMinutes=15


[Display]

; 각 TOP 목록에 출력할 최대 개수입니다.
TopCount=10

; 미리보기에서 표시할 방송 제목의 최대 글자 수입니다.
; 길어질 경우 자동으로 ... 처리됩니다.
PreviewTitleWidth=18

; 분석 결과 파일에 상세 로그를 포함할지 설정합니다.
; False로 설정하면 TOP 결과만 저장합니다.
SaveDetailLog=True
```

## 설정 검증

프로그램 실행 시 모든 설정값을 검증한다.

검증 대상

* 설정값 존재 여부
* 자료형
* 허용 범위
* 문자열 길이
* 빈 문자열 여부

---

## 설정 복구

유효하지 않은 설정값이 발견되면 프로그램은 종료되지 않는다.

잘못된 항목만 기본값으로 변경한 뒤 사용자에게 변경 사유를 안내한다.

예시

```
다음 설정값이 올바르지 않아 기본값으로 복원했습니다.

MaxVods = 감자
→ 정수만 입력 가능합니다.
→ 기본값(30)으로 변경되었습니다.

MovingAverageMinutes = 100
→ 허용 범위(1~60)를 벗어났습니다.
→ 기본값(15)으로 변경되었습니다.
```

모든 검증이 끝난 후 수정된 내용은 설정 파일에 자동 저장된다.

## 설정 파일 형식 검증

설정 파일에는 형식 버전을 포함한다.

```
ConfigFormat=1
```

프로그램이 요구하는 형식과 다를 경우 현재 버전에서 사용할 수 없는 설정 파일로 판단한다.

이 경우

* 기존 설정 파일을 초기화한다.
* 최신 형식의 설정 파일을 새로 생성한다.

안내 메시지

```
설정 파일 형식이 현재 버전과 다릅니다.

최신 형식으로 초기화했습니다.
```

---

## 구현 구조

설정 시스템은 다음과 같이 구성한다.

```
create_settings_if_not_exists()
        │
        ▼
load_settings()
        │
        ▼
validate_settings()
```

각 함수 역할

* create_settings_if_not_exists()

  * 설정 파일 존재 확인
  * 최초 실행 시 기본 설정 생성

* load_settings()

  * 설정 파일 읽기

* validate_settings()

  * 형식 검증
  * 항목별 유효성 검사
  * 잘못된 값 복구
  * 필요 시 설정 파일 저장

---

## 기대 효과

* 최초 실행 시 별도 설정 없이 바로 사용 가능
* 사용자가 메모장으로 쉽게 설정 수정 가능
* 잘못된 설정으로 프로그램이 종료되지 않음
* 버전 변경 시 설정 파일 자동 갱신
* 향후 설정 항목 추가 및 유지보수가 용이함
