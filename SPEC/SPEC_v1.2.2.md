# ChzzkVodChat v1.2.2 Specification

작성일 : 2026-06-28

---

# 1. 변경 목적

사용자가 프로그램을 여러 번 실행해야 하는 불편함을 줄이고,
원하는 시간대의 채팅만 분석할 수 있도록 분석 기능을 개선한다.

---

# 2. 변경 사항

## 2.1 메인 메뉴 복귀 기능 수정

### 문제점

채팅 수집 완료 후 프로그램이 즉시 종료된다.

다른 기능을 사용하려면 프로그램을 다시 실행해야 한다.

### 개선

채팅 수집 완료 후 프로그램을 종료하지 않고 메인 메뉴로 복귀한다.

모든 기능은 작업 완료 후 메인 메뉴로 돌아가도록 동작을 통일한다.

### 동작 흐름

메인 메뉴

↓

채팅 수집

↓

수집 완료

↓

메인 메뉴 복귀

---

## 2.2 채팅 분석 모드 UI 변경

### 변경 목적

기존 다중 파일 입력 기능을 제거하고,
단일 채팅 파일의 원하는 시간대만 분석할 수 있도록 변경한다.

### 기존 방식

파일명 입력

↓

키워드 입력

↓

분석

### 변경 방식

파일명 입력

↓

분석 시작 시간 입력

(미입력 시 00:00:00)

↓

분석 종료 시간 입력

(미입력 시 영상 종료 시간)

↓

키워드 입력

↓

영상 정보 확인

(Y/N)

↓

분석 시작

---

### 입력 항목

#### 1. 파일명

분석할 JSON 또는 TXT 파일의 파일명을 입력합니다.
확장자는 입력하지 않아도 됩니다.

---

#### 2. 시작 시간

입력 형식

HH:MM:SS

예시

00:15:00

미입력 시

00:00:00

---

#### 3. 종료 시간

입력 형식

HH:MM:SS

예시

02:30:00

미입력 시

영상 종료 시간

---

#### 4. 키워드

분석할 키워드 1개 입력

예시

ㅋㅋ

---

#### 5. 영상 정보 확인

분석 예정 정보를 출력한다.

예시

=========================================
파일명
0627.txt

분석 구간
01:30:00 ~ 02:00:00

키워드
ㅋㅋ

예상 분석 대상
약 30분
=========================================

계속 진행하시겠습니까?

(Y/N)

Y

→ 분석 시작

N

→ 메인 메뉴 복귀

---

## 2.3 시간 입력 검증

다음 입력은 오류 처리한다.

- HH:MM:SS 형식이 아닌 경우
- 시작 시간이 종료 시간보다 늦은 경우
- 영상 길이를 초과하는 시간 입력
- 음수 시간 입력

오류 발생 시 안내 메시지를 출력한 뒤 해당 입력을 다시 받는다.

---

# 3. 제거 기능

## 다중 파일 입력 모드 제거

기존 여러 txt 파일을 동시에 분석하는 기능을 제거한다.

채팅 분석은 항상 하나의 txt 파일만 대상으로 수행한다.

---

# 4. 기대 효과

- 프로그램 재실행 없이 연속 작업 가능
- 원하는 시간대만 분석하여 분석 시간 단축
- 긴 방송에서도 필요한 구간만 빠르게 확인 가능
- 입력 과정 단순화
- 사용자 편의성 향상
