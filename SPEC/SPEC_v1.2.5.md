# ChzzkVodChat v1.2.5 Specification

작성일 : 2026-07-01

# 버전 목표

v1.2.5는 채팅 분석 알고리즘을 개선하여 TOP 요약의 정확도를 높이고,
프로그램의 사용성과 데이터 관리 방식을 개선하는 것을 목표로 한다.

---

# 변경 사항

## 1. 채팅 분석 알고리즘 개선

### 변경 목적

기존 TOP 요약 생성 과정에서 병합 후 추가 병합이 정상적으로 이루어지지 않는 문제를 수정하고,
사용자가 설정한 TOP 개수를 더욱 정확하게 유지하도록 알고리즘을 개선한다.

### 변경 내용

#### 1-1. TOP 후보 계산 확장

TOP 계산 시 설정된 개수의 3배까지 미리 계산한다.

예시

- TOP 5 → 후보 15개 계산
- TOP 10 → 후보 30개 계산
- TOP 20 → 후보 60개 계산

---

#### 1-2. 병합 수행

현재 TOP 순위만을 대상으로 시간대 병합을 수행한다.

연속되는 시간대가 존재할 경우 하나의 구간으로 병합한다.

---

#### 1-3. 병합 후 순위 보충

병합으로 인해 TOP 개수가 감소할 경우,
다음 후보 순위를 현재 TOP에 추가한다.

추가된 순위는 즉시 병합 대상에 포함된다.

---

#### 1-4. 병합 규칙

- 병합은 현재 TOP 순위 내에서만 수행한다.
- 병합으로 추가된 순위 역시 병합 대상이다.
- 병합된 구간은 추가 병합 대상에서 제외한다.
- 사용자가 설정한 TOP 개수가 될 때까지 병합 및 순위 보충을 반복한다.

예시

```
TOP5

1 + 2 + 6
3 + 8
4 + 7
5
9
```

---

### 기대 효과

- 병합 누락 버그 해결
- TOP 개수 유지
- 더 자연스러운 하이라이트 구간 생성
- 기존 병합 로직의 안정성 향상

## 2. 메인 메뉴 종료 안내 추가

### 변경 목적

ESC 종료 기능을 사용자가 쉽게 확인할 수 있도록 안내를 추가한다.

### 변경 내용

메인 메뉴 하단에 다음 문구를 표시한다.

```
ESC를 누를 시 프로그램이 종료됩니다.
```

---

### 기대 효과

- 종료 기능 인지 향상
- 사용자 편의성 개선

---

## 3. JSON 저장 방식 개선

### 변경 목적

채팅 분석에서 실제 사용하는 데이터는 시간과 채팅 내용이므로,
불필요한 JSON 파일 저장을 제거하여 프로그램 구조를 단순화한다.

### 변경 내용

- 채팅 수집 과정에서는 기존과 동일하게 JSON 데이터를 사용한다.
- 수집 완료 후 JSON 데이터를 기반으로 TXT 파일을 생성한다.
- TXT 파일이 정상적으로 생성되면 JSON 파일은 자동 삭제한다.
- TXT 생성에 실패한 경우 JSON 파일은 삭제하지 않는다.
- 이후 채팅 분석 기능은 TXT 파일만을 사용한다.

### 기대 효과

- 저장 공간 절약
- 파일 관리 단순화
- 분석 입력 형식 통일
- 프로젝트 구조 개선
