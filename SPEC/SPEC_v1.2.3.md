# CHZZK VOD CHAT ANALYZER v1.2.3 Specification

## 개요

v1.2.3은 **TOP 랭킹의 직관성과 분석 품질 향상**을 목표로 하는 업데이트이다.

기존에는 동일한 장면이 여러 순위를 차지하는 문제가 있었으며, 반복 키워드로 인해 일부 채팅의 영향력이 과도하게 반영되는 경우가 있었다.

이번 버전에서는 **클립 병합 기능**, **키워드 반복 제한**, **TOP 후보 선정 기준 개선**, **UI 편의성 개선**을 적용한다.

---

# 1. TOP 랭킹 클립 병합

## 목적

동일한 장면이 여러 순위를 차지하는 현상을 방지하여 실제 하이라이트 기준으로 TOP 랭킹을 제공한다.

### 기존

```
1. [02:50:00]
5. [02:51:00]
```

### 변경

```
1. [02:50:00 ~ 02:51:00]
```

---

## 병합 규칙

* TOP 랭킹 계산 후 병합을 수행한다.
* 랭킹이 높은 항목부터 순차적으로 처리한다.
* 현재 구간과 ±1분 차이인 시간대를 병합 대상으로 판단한다.
* 병합이 발생하면 다시 인접 시간대를 검사한다.
* 병합은 최대 **3개의 시간대**까지만 허용한다.
* 병합된 항목은 랭킹에서 제거하며 다음 순위가 자동 승격된다.
* 최종 결과는 항상 TOP10을 유지하도록 다음 후보를 추가한다.

---

## 병합 예시

초기 랭킹

```
1. 02:51
4. 02:53
9. 02:50
11. 02:52
```

↓

최종 결과

```
1. 02:50 ~ 02:52
4. 02:53
```

(02:53은 최대 병합 개수 제한으로 인해 유지)

---

# 2. TOP 출력 개선

병합된 구간은 시간순으로 출력하여 직관성을 높인다.

## 출력 예시

```
====================================
채팅량 TOP10
====================================

1. [02:50:00 ~ 02:52:00]

   [02:50]
   채팅 132 | 키워드 20 (15.2%) ▲   | 평균 +153% ▲▲▲

   [02:51]
   채팅 103 | 키워드 35 (34.0%) ▲▲▲ | 평균 +78% ▲▲

   [02:52]
   채팅 90 | 키워드 10 (11.1%) ▲   | 평균 +41% ▲
```

## 출력 규칙

* 병합된 시간대는 항상 **시간순(오름차순)** 으로 출력한다.
* 기존 통계(채팅 수, 키워드 수, 키워드 비율, 이동평균)는 그대로 출력한다.
* 병합 구간의 합계는 별도로 출력하지 않는다.

---

# 3. 키워드 반복 카운트 제한

## 목적

동일한 채팅에서 특정 키워드를 과도하게 반복하는 경우 분석 결과가 왜곡되는 현상을 방지한다.

### 기존

```
ㅋㅋㅋㅋㅋㅋㅋㅋㅋㅋㅋㅋㅋㅋ
```

↓

```
키워드 12회
```

### 변경

```
키워드 5회
```

## 적용 규칙

* 한 개의 채팅에서 동일 키워드는 최대 **5회**까지만 카운트한다.
* 초과되는 반복은 무시한다.
* 다른 채팅에는 영향을 주지 않는다.

---

# 4. TOP 최소 채팅 컷 변경

## 변경 내용

기존

```
사용자 지정 최소 채팅 수
```

↓

변경

```
전체 평균 채팅 수
```

## 적용 규칙

* TOP 랭킹은 **전체 평균 채팅 수 이상**인 시간대만 후보로 사용한다.
* 평균보다 채팅량이 적은 시간대는 TOP 랭킹에서 제외한다.
* 이동평균 증가율 및 기타 분석에는 영향을 주지 않는다.

---

# 5. 메인 메뉴 종료 단축키

## 기능

메인 메뉴에서 **ESC** 입력 시 프로그램을 즉시 종료한다.

## 적용 범위

* 메인 메뉴
* 입력 대기 상태

## 동작

```
ESC 입력
    ↓
프로그램 종료
```

---

# 변경 사항 요약

## 신규 기능

* TOP 랭킹 연속 시간대 병합
* 병합 후 인접 시간대 재검사
* 최대 3개 시간대 병합 제한
* 병합 구간 시간순 출력
* 메인 메뉴 ESC 종료

## 개선 사항

* 병합된 구간에 분별 통계 출력
* 한 채팅당 동일 키워드 최대 5회만 카운트
* TOP 최소 채팅 컷을 전체 평균 채팅 수 기준으로 변경

## 제외 사항

* 이동평균 계산 방식 변경
* 병합 구간 합계 출력
* 키워드 분석 알고리즘 변경(반복 제한 제외)
